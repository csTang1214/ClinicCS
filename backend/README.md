# Backend

Minimal Express.js backend server with TypeScript support.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs the server with hot reload on http://localhost:5000

## Build

```bash
npm run build
```

Compiles TypeScript to `dist/` folder.

## Production

```bash
npm run start
```

Runs the compiled server.

## API

- `GET /api/health` - Health check endpoint
